# KCET Predictor

AI-powered KCET college predictor and counselling assistant.

## Features

- Rank Prediction
- Category Filtering
- Branch Filtering
- Bangalore College Recommendations
- Safe / Likely / Dream Classification
- Excel Report Generation

## Dataset

- KEA UGCET 2025 Round 3
- 229 Colleges
- 110 Branches
- 10,949 Cutoff Records
